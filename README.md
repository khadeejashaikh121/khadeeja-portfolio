## Technical Stack

- Built with [Astro](https://astro.build/)
- Styled with [Tailwind CSS](https://tailwindcss.com/)
- Icons from [Tabler Icons](https://tabler.io/icons)
- TypeScript for type-safe configuration

## Local Development

```bash
npm install
npm run dev
```

## Deployment

This portfolio can be deployed to any static hosting service such as Netlify, Vercel, GitHub Pages, or Cloudflare Pages.
